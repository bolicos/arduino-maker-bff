# MongoDB

Run this below command on root of the project:

```shell
docker-compose -f mongodb.yml up
```

Wait for it to initialize completely, and visit:

- http://swarm-ip:8081
- http://localhost:8081
- http://0.0.0.0:8081
- http://host-ip:8081 (as appropriate).
